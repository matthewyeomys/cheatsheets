\section{Dynamic Programming}
Everything is a table!
\paragraph{Optimal Sub-Structure}
Optimal solution can be contructed from optimal solutions to smaller subproblems\\
\textcolor{Bittersweet}{Doesn't always exist}
\paragraph{DP Recipe}
\begin{itemize}
  \item Identify optimal substructure
  \item Define subproblems
  \item Solve problem using subproblems
  \item Write pseudocode
\end{itemize}
\paragraph{DP Analysis}
\begin{itemize}
  \item Count subproblems
  \item Figure out total time to solve all subproblems
\end{itemize}
